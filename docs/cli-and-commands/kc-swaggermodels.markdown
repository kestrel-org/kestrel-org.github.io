# kc swaggerModels

## Feature

Create new swagger model for the swagger api from the database used in the project and put them in a folder named swaggerModels

## Usage

{% highlight bash %}
$ kc swaggerModels
$ kc swagm
{% endhighlight %}

## Example

{% highlight bash %}
$ kc swagm
{% endhighlight %}
