# kc doc

## Feature

Generate documentation for the angular project

## Usage

{% highlight bash %}
$ kc doc [options]
{% endhighlight %}

## Options

| Option         | Description                             |
| :------------- | :-------------------------------------- |
| `--serve` `-s` | serve documentation after generating it |

## Example

{% highlight bash %}
$ kc doc
{% endhighlight %}
